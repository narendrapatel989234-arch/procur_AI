1. User Goal Understand current state of active requests and identify what needs attention today Start a new procurement request quickly and accurately with minimal manual effort Confirm PR is accurate and complete, then submit with confidence Know PR is correctly classified and understand where it sits in approval without chasing Select best vendor from AI recommendations confidently and submit for final approval Confirm PO is correctly generated and procurement lifecycle is fully closed Produce a comprehensive RFP and get it approved for distribution Collect all vendor proposals and ensure AI extraction is complete and accurate Compare all vendors using AI evaluation and make a well-supported vendor selection Finalize a compliant, contract-ready SoW with minimal manual effort and submit for approval Confirm PO generated after signed SoW, verify sync, and close lifecycle cleanly

User Actions UserLog in → land on Dashboard → scan active requests table → review KPI cards → identify priority items UserClick New Request → select entry mode (chat / form / doc upload) → describe need → upload documents UserReview AI PR draft → edit incorrect fields → resolve duplicate alert → confirm and submit PR UserReview AI classification + reasoning → monitor lifecycle progress → respond to manager clarification if returned UserReview AI vendor list → expand justification per vendor → select preferred vendor → submit for approval UserMonitor approval outcome → open PO tab → verify line items + vendor details → confirm ERP sync UserReview AI RFP draft → edit sections → configure scoring criteria → manage vendor shortlist → submit for approval UserUpload received vendor proposals → link each to vendor → monitor extraction status → handle failures UserReview AI evaluation report → inspect comparative grid → adjust weights if needed → review negotiation brief → finalize vendor UserReview AI SoW variants → select/merge template → edit sections → resolve compliance findings → submit for approval UserUpload signed SoW → review auto-generated PO summary → verify ERP sync status
3. System Interactions AI AutoRole-aware Dashboard loads with personal KPIs pre-filtered → SLA at-risk requests surfaced → pending actions ranked by urgency AI AutoParses chat intent → asks contextual clarification → triggers doc extraction on upload → pre-fills form fields → shows confidence indicators AI AutoValidates field completeness → flags missing/low-confidence fields → checks for duplicate PRs → generates PR with unique ID → triggers workflow orchestration AI AutoClassifies PR as Routine or Complex with reasoning → routes to correct approval chain → updates lifecycle stage + SLA status in real time AI AutoSurfaces recommended vendors with confidence scores + citations immediately after PR approval → runs inline compliance check → submission triggers approval workflow AI AutoManager approval triggers PO generation → ERP/SharePoint sync runs automatically → lifecycle status updates to Closed on successful sync AI AutoGenerates RFP draft from approved PR context immediately after complex classification → section writing assist on demand → version history tracked → submission triggers approval AI AutoTriggers extraction immediately on each upload → processes proposal data → updates status per vendor (Processing / Complete / Failed) → flags failures with error reason AI AutoGenerates complete scored evaluation report after all extractions complete → produces comparative analysis → flags low-confidence scores → vendor finalization triggers SoW generation automatically AI AutoGenerates multiple SoW variants immediately after vendor finalization → runs compliance validation automatically → flags issues with severity ratings → suggests fallback language per clause AI AutoSigned SoW upload triggers PO generation immediately → validates PO data → runs ERP/SharePoint sync → updates lifecycle status to Closed on successful sync

8. Pain Points
  • No clear priority order on Dashboard
  • Overwhelming for new users
  • No guided empty state
  • Uncertainty between entry modes
  • Unclear required fields upfront
  • Anxiety about AI extraction accuracy
  • Hard to spot AI errors at a glance
  • No signal for which fields need review
  • Duplicate submission risk
  • No visibility into approval delay reasons
  • Unclear SLA countdown
  • No easy way to dispute misclassification
  • No reasoning behind recommendations
  • Unable to add vendor not in system
  • Compliance result unclear
  • ERP sync failure with no clear recovery
  • No visible confirmation of full lifecycle closure
  • Generic AI sections need heavy rewriting
  • No guidance on evaluation criteria best practices
  • Shortlist limited to system vendors only
  • Manual upload friction for multiple proposals
  • No vendor response tracking
  • Extraction failure with no recovery path
  • Black-box scoring without explanations
  • Low-confidence flags without actionable guidance
  • No weight adjustment without IT support
  • Compliance findings hard to action without legal knowledge
  • Clause library hard to search
  • Version confusion with multiple drafts
  • ERP sync failure blocking closure
  • No clear full-lifecycle completion confirmation
  • PO data errors discovered only after generation
